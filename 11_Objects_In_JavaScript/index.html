Data Types in javascript

2 Types of data types in js

  1. Primitive(Value Types)
  2. Non Primitive(Reference Types)

Primitive data types categorized into 7

  1. number
  2. string
  3. boolean
  4. null
  5. undefined
  6. bigInt
  7. symbol

There is only 1 type of non primitive data type i.e. called object

objects are further divided into 3 categories:

  • Object Literals
  • Arrays
  • Functions
